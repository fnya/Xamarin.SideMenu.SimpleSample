# Xamarin.SideMenu.SimpleSample

このプロジェクトは、 [Xamarin.SideMenu](https://github.com/TheEightBot/Xamarin.SideMenu) の最低限のシンプルなサンプルになります。

大元の `Xamarin.SideMenu` はサンプルがついているのですが、内容を読み解くのが難しいのでこのプロジェクトを作成しました。
